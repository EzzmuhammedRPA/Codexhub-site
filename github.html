ايه هو GitHub؟

GitHub هو خدمة استضافة لمستودعات Git بتسهل العمل الجماعي، مراجعة الكود (Pull Requests)، إدارة القضايا (Issues)، وContinuous Integration. GitHub مش هو Git — هو بس المكان اللي بترفع عليه المستودعات علشان يشاركها فريقك أو العالم.

تدفق عمل شائع (Fork / PR)

  1. اعمل Fork للمشروع لو مش عندك صلاحيات الكتابة.
  2. انسخ (clone) للمشروع على جهازك: git clone <repo-url>
  3. اعمل فرع جديد: git checkout -b feature/my-feature
  4. اعمل تغييرات، git add، git commit ثم git push.
  5. افتح Pull Request من الفرع بتاعك للمشروع الأصلي واطلب مراجعة.

نصايح للمبتدئين

  • اكتب رسائل Commit واضحة.
  • استخدم فروع منفصلة لكل ميزة أو تصليح.
  • اعمل Code Review قبل الدمج.
