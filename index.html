What's git ?

Git هو نظام تحكم في الإصدارات (Version Control System) مصمّم لإدارة تغييرات أكواد المشاريع بسرعة وكفاءة. بيخليك تقدر ترجع لأي نسخة قبل كده، تشتغل على فروع (branches)، وتندمج التغييرات بسهولة.

  • موزّع: كل مطوّر عنده نسخة كاملة من المستودع.
  • سريع: معظم العمليات تتم محليًا.
  • متعدد الفروع: بيشجّع العمل على فروع منفصلة لكل ميزة.

خطوات سريعة

  1. تهيئة مستودع: git init
  2. إضافة ملفات: git add .
  3. حفظ نقطة: git commit -m "first commit"
